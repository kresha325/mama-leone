# Mama Leone – Ristorante Pizzeria

Website moderne me menu dixhitale për **Mama Leone** në Meldorf, Gjermani.

## Karakteristikat

- Design elegant me temë të errët (chalkboard) dhe ngjyra italiane
- Menu dixhitale e plotë me të gjitha kategoritë
- Kërkim i gjërave dhe filtrim allergenesh
- Seksione: Hero, Über uns, Speisekarte, Kontakt
- Hartë Google Maps e integruar
- Responsive për mobile dhe desktop

## Të dhënat e biznesit

- **Adresa:** Südermarkt 7 • 25704 Meldorf
- **Telefon:** 0 48 32 / 97 87 888
- **Website:** www.mama-leone.de
- **Orari:** 11:30–14:30 & 17:30–22:00 (të mbyllur të martën)

## Instalimi

```bash
npm install
npm run dev
```

Hapni [http://localhost:3000](http://localhost:3000).

## Build për production

```bash
npm run build
npm start
```

## Teknologjitë

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS
